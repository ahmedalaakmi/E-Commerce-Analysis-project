# 🛒 E-Commerce Data Analysis — Excel

> "A little practice beats a lot of theory."
> This is my first Data Analysis project using Excel,
> focused on an E-Commerce domain.

---

## 📁 Dataset Overview

The dataset contains **8 Tables:**

| Table | Description |
|---|---|
| Customers | Customer information |
| Employees | Employee records |
| Orders | Order transactions |
| Order Details | Detailed order lines |
| Products | Product catalog |
| Categories | Product categories |
| Shippers | Shipping companies |
| Suppliers | Supplier information |

---

## 🧹 Data Cleaning — Power Query

Found and handled **3 main issues:**

- **Null Values** — handled based on each case separately
- **Duplicates** — detected fully duplicated rows across all columns
- **Data Type Errors** — fixed to avoid modeling issues

---

## 🧩 Data Modeling

- Identified **Fact Tables** vs **Dimension Tables**
- Defined correct **Relationship types** between tables
- Loaded model into **Data Model** to keep sheets clean
